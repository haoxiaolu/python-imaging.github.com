Python Imaging

Welcome to Python Imaging.

This is the home of Pillow the popular Python Imaging Library (PIL) fork. If you have ever worried or wondered about the future of PIL, please stop! We are here to save the day.

Installation

You can install Pillow with pip:

$ pip install Pillow

Or easy_install (particularly useful for installing eggs since pip does not support them):

$ easy_install Pillow

Or download the compressed archive from PyPI, extract it, and inside it run:

$ python setup.py install

Documentation

Our documentation is hosted on ReadTheDocs.

Mailing List

We discuss Pillow and PIL on the Image-sig list.

IRC

We chat about Pillow and PIL in real time on irc.freenode.net in the #pil channel. We also maintain a log of our conversations.
